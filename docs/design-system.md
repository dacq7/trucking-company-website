# Design System
Trucking Insurance & Compliance Website

This document defines the visual design rules for the website to ensure a consistent and professional interface.

---

# Design Style

Professional, modern and trustworthy.

The design must communicate reliability, safety and professionalism for trucking companies operating in the United States.

The interface should feel similar to modern logistics, transportation, and financial service websites.

Avoid flashy or playful styles.

Focus on clarity, readability, and trust.
